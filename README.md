## 1. 프로젝트 소개

Active + Verve : 활동적인, 적극적인 + 열정, 활력, 생기

활동적이고 열정적인 분위기를 조성하며, 소통과 함께 성장하는 운동 경험을 제공하는 커뮤니티!
